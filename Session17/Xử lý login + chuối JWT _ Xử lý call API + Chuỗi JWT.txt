	* Xác thực với username và password:

		+ Xác thực thành công sẽ trả về đối tượng: Authentication
		+ Sử dụng đối tượng này gọi phương thức: getPrincipal() -> Trả về CustomUserDetails
	
	+ Phải cài đặt lớp: JWTProvider thực hiện các nhiệm vụ:
		+ Sinh ra chuỗi token từ username
		
	-> Kết quả trả sau phần 2:

Phần 3:
	Xử lý call api + chuỗi jwt

	1. Cài đặt JWTAuthFilter:
		- Là 1 filter (tự chạy)
		- Kế thừa OncePerRequestFilter (Mỗi lần request lại xác thực lại)
		
		Các bước thực hiện:
		1.1 Tạo class JWTAuthFilter extends OncePerRequestFilter
		1.2 Override method: doFilterInternal(...):
			1.2.1 Tách chuỗi token từ request
			1.2.2 Cài đặt phương thức validateToken(String token) trong JWTProvider
			1.2.3 Cài đặt hàm getUsernameFromToken(Strong token) trong JWTProvider
			1.2.4 Xử lý tiếp trong doFilterInternal
					- Lấy username từ token
					- Sử dụng UserDetailsService gọi hàm loadUserByUsername() trả về CustomUserDetails
					- Tạo ra Authentication
					- Gán vào SecurityContextHolder
		
	2. Cài đặt 1 lớp xử lý lỗi khi chưa xác thực
		Cài đặt lớp JWTAuthenticationEntryPoint kế thừa AuthenticationEntryPoint
	
	3. Thêm 1 số cấu hình:
		- Cấu AuthenticationProvider
		- Thêm cấu hình vào SecurityFilterChain
