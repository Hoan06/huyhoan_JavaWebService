Bước 1: Khởi tạo môi trường trên PostmanMở ứng dụng Postman trên máy tính.Tại thanh bên trái (Sidebar), nhấn vào tab Collections $\rightarrow$ chọn biểu tượng dấu cộng + $\rightarrow$ Đặt tên Collection mới là BaiTap_WebService.Bước 2: Thực hiện các Request theo yêu cầu1. GET - Lấy tất cả bài viếtTên Request trong Collection: 1. Get All PostsPhương thức (Method): GETĐường dẫn (URL): https://jsonplaceholder.typicode.com/postsHành động: Nhấn nút Send.Kết quả nhận được:Mã trạng thái (Status Code): 200 OKPhần Thân (Body): Trả về một mảng JSON chứa 100 object bài viết (từ id: 1 đến id: 100).Tab Headers của Response: Xuất hiện cặp key-value: Content-Type: application/json; charset=utf-8.📸 [Ảnh chụp màn hình đề xuất]: Chụp toàn bộ giao diện Postman hiển thị URL GET, nút Send, Mã 200 OK và một phần danh sách JSON ở phía dưới.2. GET - Lấy một bài viết cụ thể (id = 1)Tên Request trong Collection: 2. Get Post By IDPhương thức (Method): GETĐường dẫn (URL): https://jsonplaceholder.typicode.com/posts/1Hành động: Nhấn nút Send.Kết quả nhận được:Mã trạng thái (Status Code): 200 OKPhần Thân (Body): Chỉ trả về duy nhất 1 object JSON của bài viết có id: 1 với cấu trúc:JSON{
"userId": 1,
"id": 1,
"title": "sunt aut facere repellat provident occaecati excepturi optio reprehenderit",
"body": "quia et suscipit\nsuscipit..."
}
POST - Tạo bài viết mới(Do phần đề bài của bạn bị khuyết đoạn JSON nhập vào, dưới đây là đoạn JSON chuẩn theo cấu trúc của JSONPlaceholder để bạn sử dụng)Tên Request trong Collection: 3. Create New PostPhương thức (Method): POSTĐường dẫn (URL): https://jsonplaceholder.typicode.com/postsCấu hình phần Body: 1. Chuyển sang tab Body (nằm ngay dưới thanh nhập URL).2. Chọn kiểu dữ liệu là raw.3. Ở menu thả xuống ngoài cùng bên phải, chọn định dạng là JSON.4. Nhập đoạn dữ liệu sau vào khung text:json { "title": "Bài tập Web Service của Huy Hoàn", "body": "Nội dung thực hành kiểm thử API bằng công cụ Postman.", "userId": 1 } Hành động: Nhấn nút Send.Kết quả nhận được:Mã trạng thái (Status Code): 201 CreatedPhần Thân (Body): Trả về object vừa tạo kèm theo một ID mới do server cấp phát ("id": 101).📸 [Ảnh chụp màn hình đề xuất]: Chụp rõ phần cấu hình tab Body (chọn raw $\rightarrow$ JSON) và mã trạng thái 201 Created ở dưới.4. DELETE - Xóa một bài viết (id = 1)Tên Request trong Collection: 4. Delete PostPhương thức (Method): DELETEĐường dẫn (URL): https://jsonplaceholder.typicode.com/posts/1Hành động: Nhấn nút Send.Kết quả nhận được:Mã trạng thái (Status Code): 200 OKPhần Thân (Body): Trả về một object rỗng {} (do đây là API giả lập - Mock API, nên hệ thống chỉ mô phỏng lại phản hồi thành công chứ không thực sự xóa dữ liệu trong database).
